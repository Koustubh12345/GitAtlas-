📝 Changelog (v1.0)

| Category | Updates & Features |
| :--- | :--- |
| **Smart Feed** | Added auto-algorithm to fetch personalized recommendations based on saved and searched repos. |
| **UI / UX** | Implemented Liquid Glass UI, heavy blur effects, and Material X design language. |
| **Repo Viewer** | Read repo code natively in-app with desktop view toggle, web browser redirect, and sharing options. |
| **Discovery** | Added dedicated tabs for Trending repositories, GitHub Developers, and advanced Searching. |
| **File Explorer** | Native file manager to view downloaded repository source code locally directly within the app. |
| **Core Actions** | Support for direct repository downloads and offline bookmarking/saving. |
| **System** | Full Android 7.1+ support built on a clean MVVM architecture. |
